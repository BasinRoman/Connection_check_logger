Небольшая утилитка, которая помогала мне в работе системного администратора. 
Файл ping_check.bat собирает ping до нужного узла и записывает данные в log.txt 
Скрипт main.py парсит этот файл и выдает ТОЛЬКО пропуски соединения. 
В дальнейшем благодаря таймингам разрывов соединения можно анализировать проблему.
